# CV & Portfolio Page

A professional CV and portfolio website built with Tailwind CSS.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Built with Tailwind CSS for a clean, professional look
- **Easy to Customize**: Simple HTML structure with placeholder content
- **Navigation**: Smooth scrolling navigation to different sections
- **Sections Included**:
  - Hero section with profile photo and introduction
  - Education background
  - Project showcase
  - Skills overview
  - Affiliations & Memberships (optional)
  - Research & Publications (optional)

## Getting Started

### Without Building (Quick Start)
Simply open `index.html` in your browser. The page uses the Tailwind CDN, so no build process is required.

### With Tailwind Build Process

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development mode:**
   ```bash
   npm run dev
   ```
   This will watch your CSS files and rebuild on changes.

3. **Build for production:**
   ```bash
   npm run build
   ```

## Customization

1. **Update your information:**
   - Replace "Your Full Name" with your actual name
   - Update the professional title/tagline
   - Add your contact email
   - Update social media links

2. **Add your photo:**
   - Replace the placeholder image URL with your own photo
   - Keep it professional and clear

3. **Fill in your details:**
   - Education section: Add your degrees and institutions
   - Projects section: Add your actual projects with descriptions
   - Skills section: List your technical skills
   - Affiliations: Add any memberships or affiliations
   - Research: Add publications or research work

4. **Modify colors:**
   - Edit the gradient colors in the hero section (change `from-blue-600 to-blue-800`)
   - Update accent colors throughout the page (currently blue-600)
