# Jordan Ereahn Tubianosa - CV & Portfolio

My professional CV and portfolio website showcasing my educational background, technical skills, and projects. Built with HTML5 and Tailwind CSS to create a clean, modern, and fully responsive design.

## About This Project

This is my personal portfolio website where I present my academic journey and professional experience. I created this to showcase my skills as a BS Information Technology student and demonstrate my ability to build responsive web applications.

## Key Features

- **Fully Responsive Design**: Works perfectly on mobile phones, tablets, and desktop computers

- **Professional Layout**: Two-column grid design that organizes information clearly
- **Sections Included**:
  - Profile section with photo and introduction
  - Education background (4 educational institutions)
  - Project showcase (HitchPH, Personal Portfolio)
  - Technical skills with proficiency levels

  - Contact information and social media links

## How to Run This Project

If you want to view or modify this portfolio locally:

1. **Install the required dependencies:**
   ```bash
   npm install
   ```

2. **Run in development mode to watch for changes:**
   ```bash
   npm run dev
   ```
   This will monitor my CSS files and rebuild automatically when I make changes.

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **View the portfolio:**
   Open `index.html` in your browser to see the final result.

## Design Decisions

I chose Tailwind CSS because it allows me to quickly build a responsive, professional-looking website without writing extensive custom CSS. The two-column layout on desktop provides clear organization of my education, projects, and skills, while the single-column mobile view ensures the portfolio is fully accessible on all devices.

The color scheme uses a blue gradient header (#4a90a4 to #357a8f) to create a professional appearance and guide attention to my name and contact information.
